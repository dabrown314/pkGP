# pkGP
Code for projected kernel Gaussian processes

This project contains code implementing the projected kernel Gaussian process (pkGP) proposed in the following paper:

Brown, D. A., Kiessler, P., and Nicholson, J., "A kernel-based approach for Gaussian process modeling with functional information," SIAM/ASA Journal on Uncertainty Quantification 14, 508-533, 2026.

Please cite this paper if you use this code in your own work.
